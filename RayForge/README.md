# RayForge

RayForge is a retro-style 3D raycasting engine written in C# using Windows Forms and GDI+.

![Platform](https://img.shields.io/badge/.NET-8.0-blue)
![Language](https://img.shields.io/badge/C%23-WinForms-purple)
![Status](https://img.shields.io/badge/Status-Active-success)

---

## Features

- Classic Wolfenstein-like raycasting
- Real-time first-person rendering
- Collision detection
- Smooth player movement
- Dynamic minimap
- Distance-based wall shading
- Adjustable field of view

---

## Controls

| Key | Action |
|-----|--------|
| W / ↑ | Move forward |
| S / ↓ | Move backward |
| A / ← | Rotate left |
| D / → | Rotate right |
